To build the server-side component of a “movies” web application.
The web application will provide users with access to information
about different movies, directors, and genres.
Users will be able to sign up, update their personal information,
and create a list of their favorite movies.

Endpoints executed in this program

In this api we have a number of actions that can be executed

Request

Get a whole list of movies


URL- /movies
HTTP method- GET
Request data format- none
This JSON object will return data about all movies
example of returned results
{ title: 'The Green Book', director: 'Peter Farrelly' },

Request

Get data about a specific movie
URL- movies/title.
HTTP method- GET
Request data format- none
This JSON object will return data about a single movie and all of its pertaining data.
{ title: 'The Green Book', director: 'Peter Farrelly' },

Request

Get data about a genre for a specific movie
URL-/movies/:name
HTTP method- GET
Request data format- none
This JSON object will return data about a genre for a single movie
{ title: 'The Green Book', director: 'Peter Farrelly' genre:**** },

Request

Get data about a specific director by name
URL- /movies/:director
HTTP method- GET
Request data format- none

Request

Allow a new user to register
URL- /Users
HTTP method- POST
Request data format-{ Username: 'exampleusername' Password: 'examplepassword' Email: 'email@example.com' DOB: '00/00/0000' },
This JSON object will return either an error message saying 'missing name in request body' or send back information of the new user as shown in example of request data information
{ Username: 'exampleusername' Password: 'examplepassword' Email: 'email@example.com' DOB: '00/00/0000' },

Request

Allow an existing user to update their information
URL- /users/:Username
HTTP method- PUT
Request data format- data will come back to user to allow them to modify data { Username: 'exampleusername' Password: 'examplepassword' Email: 'email@example.com' DOB: '00/00/0000' },
This JSON object will allow the user to update their existing account information. two messages will come back saying either the informaton has been correctly entered or username could not be found

Request

Allow a user to add a movie to their favorites list
URL- /movies/:newMovie
HTTP Method- POST
Request data format- none
This JSON object will allow the user to add a movie to their favorites list.
res.send('You have successfully added a movie to your favorites list.')

Request

Allow user to remove a movie from their favorites list.
URL- /movies/:favorites
HTTP method- DELETE
Request data format- none
This JSON object will allow the user to remove a movie from their favorites list.
res.send('Successfully deleted request on removing movie from favorites list.');
